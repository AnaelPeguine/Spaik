Spaik        
                                        
* A product that lets users who are not English native speakers to enhance
                                                                
their speaking skills. Using OpenAI API.
                                                        
* Users speak a basic version of what they want to say, and the tool
                                                                
enhances it and plays the enhanced version back.
                                                        
* Use-cases include: public speaking, business presentations, and other social interactions. 


Installation


cd to  project directory
python3 main.py 
open the  localHost url in Chrome


Prerequisites
   * fastapi 
   * uvicorn 
   * openAI
   * sqlite3
   * shutil
   * pydantic


Usage


On Chrome select a speaking scenario
Tap the record button and say what is in your mind.
